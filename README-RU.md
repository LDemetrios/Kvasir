# Kvasir

Kvasir (/kwɑ'zer/) — это (неофициальный) плагин для JetBrains Platform, добавляющий поддержку для языка разметки [Typst](https://typst.app/docs).

Текущая версия в репозитории: **0.4.0**, наконец, совпадает с маркетплейсом.

![screenshot.png](screenshot2.png)

## Функционал

- Настраиваемая подсветка синтаксиса
- Включения кода (код в raw блоках подсвечивается соответственно языку)
- Моментальный предпросмотр, с возможностью подстройки под тему (цвета фона и текста передаются в `sys.inputs.kvasir-preview-background` и `-foreground` соответственно)
- Отображение ошибок компиляции
- Простые code actions (закомментировать код, автоматическая вставка закрывающих элементов)
- Форматирование (использует [typstyle](https://github.com/Enter-tainer/typstyle))
- `typc` (для кода) и `typm` (для математики) поддерживаются наравне с `typ`, со всеми перечисленными фичами.
- Настройки Kvasir доступны в Settings/Tools/Kvasir. См. [соответсвующий раздел](README-RU.md#настройки)

Плагин находится в бета-стадии, большая часть функционала ещё в разработке. 
См. "_Roadmap_" для информации о планируемых функциях,  
Не стесняйтесь открывать issue или создавать pull request'ы.

## Установка

Плагин можно установить из JetBrains Marketplace по названию или вручную с диска с помощью [архива](distributions/Kvasir-0.4.0.zip).
Старая ручная настройка `typst-shared-library` больше не требуется.

## Настройки

- **Text width**: ширина строки в символах, передаётся форматтеру. По умолчанию: `120`.
- **Tabulation size**: количество пробелов в одном уровне отступа для форматтера. По умолчанию: `4`.
- **Rainbowify brackets**: раскрашивает парные скобки и связанную пунктуацию по уровню вложенности. Включено по умолчанию.
- **Allow scrolling beyond document boundaries**: разрешает области предпросмотра уходить за края отрендеренной страницы вместо жёсткой привязки к границам документа. По умолчанию выключено.
- **Scrolling speed**: множитель скорости прокрутки предпросмотра колёсиком мыши.
- **Zoom speed**: множитель скорости масштабирования предпросмотра через Ctrl+колёсико мыши.
- **Break-glass ticket** (экспериментальная): включает привилегированный JVM-доступ из Typst-документов через upcall-функции.
  (документация когда-нибудь появится в https://github.com/LDemetrios/TyKo, может быть)
- **Suppress compilation**: отключает компиляцию предпросмотра и диагностические сообщения компилятора. По умолчанию выключено.
- **Preserved comemo cache age**: время сохранения записей кэша TyKo runtime после компиляции. При меньших значениях используется больше CPU, при больших - больше RAM. 
  По умолчанию: `20`. Для сравнения, `typst watch` использует `10`
